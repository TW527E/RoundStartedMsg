# Main Features
When Round Started automatically broadcast.

# Default config:
```yaml
round_started_msg:
  is_enabled: true
  # How long does it last
  how_long: 5
  # When Round Started Message
  on_round_started: <color=red>Go! Don't AFK!</color>
```
